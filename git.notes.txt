Create a GitHub Account

Sign Up: Go to GitHub and click on "Sign up."
Fill Out the Form: Provide a username, email, and password.
Verify Your Email: Check your email for a verification link and confirm your account.
 
Install Git
 Open your terminal and set up your username and email:

git config --global user.name "Your Name"
git config --global user.email "you@example.com"

Create a New Repository
Log In to GitHub: Go to your GitHub profile.
New Repository:

Click on the "+" icon in the top right corner and select "New repository."
Fill in the repository name and description.
Choose visibility (public or private) and click "Create repository."

Clone the Repository Locally
Get the URL: On your repository page, click the green "Code" button to copy the URL (HTTPS or SSH).

Open Terminal: Use your terminal to navigate to your desired folder.

Clone the Repository:

git clone <repository-url>
Replace <repository-url> with the URL you copied.

Add and Commit Changes

Navigate to Your Repository:

cd repository-name
Make Changes: Edit or create files in the repository.

Stage Changes:

git add .
git commit -m "Your commit message"
git push origin main
